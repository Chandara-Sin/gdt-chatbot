```sh
repository: https://github.com/Chandara-Sin/chat-bot.git
```

## Development

### Create `.env` for API

```env
VITE_API_URLL=
```

---

### Init Project

```sh
$ yarn
$ yarn set version ./.yarn/releases/*
$ yarn dlx @yarnpkg/sdks vscode
```

### Run Project

```sh
yarn dev
```

### Build Project

```sh
yarn build:local
```

### Run Unit Test

```sh
yarn test
```
